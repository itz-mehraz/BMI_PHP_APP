# BMI Calculator Application

## Overview

A web application to calculate Body Mass Index (BMI) and store user data.

## Features

- User registration and BMI calculation
- Data storage in MySQL

## Prerequisites

- PHP 7.4+
- MySQL 5.7+
- Web server (e.g., Apache, Nginx)
